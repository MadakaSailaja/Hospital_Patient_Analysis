# Hospital_Patient_Analysis
Overview
This project provides a hospital patient analytics dashboard that visualizes key healthcare metrics and patient outcomes. It is designed to support healthcare administrators, analysts, and decision-makers in monitoring performance, identifying trends, and improving patient care.
Features
- Summary metrics: total cost, average satisfaction, average length of stay, and patient count
- Interactive filters: gender and readmission status
- Visualizations:
• Gauge chart for satisfaction vs. cost
• Condition-wise cost distribution table
• Bar charts for cost by length of stay and patient outcomes
• Scatter plot for cost vs. stay duration by gender and condition
• Line chart showing readmission trends by age
• Pie and donut charts for patient demographics and condition breakdowns
• Readmission counts by condition and outcome
Purpose
- Track patient satisfaction, costs, and outcomes
- Identify areas with high readmission or costs
- Spot demographic and condition-based trends for operational improvement
Tech Stack
- Data Processing: Python / SQL
- Visualization: Power BI
- Analytics: Statistical analysis and business intelligence methods
Usage
Clone the repository and open the dashboard file in Power BI to explore the interactive visualizations.
Data sources can be customized to fit your hospital or healthcare dataset.
Project Highlight
An interactive, data-driven dashboard that transforms raw hospital data into clear, actionable insights for healthcare improvement.
